# Peak Pursuit - Climbing System
<img width="1179" height="648" alt="peak-pursuit" src="https://github.com/user-attachments/assets/b9144ee5-36ee-4e6e-969e-16acba693bfd" />

***
Peak Pursuit es el producto final de un curso de Unreal C++ donde el objetivo era crear un personaje jugable que pudiera escalar y esquivar objetos.

Contenidos relevantes:
* Componente de movimiento para escalar, extendido del Character Movement Component - Permite al personaje modificar su estado y físicas al entrar/salir del modo “Escalar”.
* Input Mapping Context (Enhanced Input System) - Para la utilización de las mismas teclas para diferentes acciones dependiendo del contexto.
* Pipeline animaciones Mixamo a Unreal (Mixamo —> Mixamo Converter —> Linked Animation) - Para la posibilidad de obtener diferentes animaciones en Mixamo, importarlas al engine, modificarlas y utilizarlas in-game.
* Control Rig - Para posicionar en tiempo de ejecución huesos dependiendo de la superficie.

***
***Engine:*** Unreal Engine v5.2

***Lenguaje:*** C++

***Plataforma:*** PC
